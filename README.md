# 🔐 Password Strength Checker
 
A lightweight, fully client-side password strength checker.
No data is sent to any server — everything runs in the browser.
 
## ✨ Features
 
- Real-time strength indicator as you type
- Entropy calculation in bits
- Estimated crack time display
- Requirement checklist (length, uppercase, numbers, special chars)
- Detects common/weak passwords
- Secure random password generator using `crypto.getRandomValues()`
- One-click copy to clipboard
